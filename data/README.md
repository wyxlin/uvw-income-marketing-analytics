# Data

This project uses the UCI Adult Income dataset.

Recommended local layout:

```text
data/
├── README.md
├── raw/
│   └── adult.data
└── processed/
```
## Download

- adult.data
- adult.names

Store `adult.data` in `data/raw/` before running the scripts

## Schema

Columns expected by the project:

1. age
2. workclass
3. fnlwgt
4. education
5. education-num
6. marital-status
7. occupation
8. relationship
9. race
10. sex
11. capital-gain
12. capital-loss
13. hours-per-week
14. native-country
15. income
